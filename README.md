🌍 Countries Explorer - REST Countries API Web App
A modern, responsive web application that fetches and displays country data from the REST Countries API. Built with React, React Router, and CSS for seamless navigation and an interactive user experience.

✨ Features
Country Listings

Displays flags, population, region, and capital for all countries

Search by country name or region

Shimmer loading effect for better UX

Country Details

Dynamic routing to view detailed country information

Responsive layout with flag, population, region, capital, and bordering countries

Error Handling

Custom 404 error page with animated UI and helpful actions

Responsive Design

Works on mobile, tablet, and desktop

Clean, modern UI with intuitive navigation

🛠️ Technologies
Frontend: React, React Router

Styling: CSS (Flexbox, Grid, Media Queries)

API: REST Countries API v3

Deployment: (Netlify)
